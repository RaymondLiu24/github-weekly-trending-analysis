# GitHub Weekly Trending Analysis

每週自動分析 GitHub 前 30 名 trending repositories，產出中文週報並累積存檔。

由 Claude Code scheduled agent 每週一凌晨自動執行。

## 報告內容

每份報告包含：
- **本週大故事** — 開源社群這週的主旋律
- **Top 10 速覽** — 排名、一句話說明、爆紅原因
- **個人精選** — 從 Top 11–30 挖出的 🧠 個人 AI 助理 / 📚 PKM 相關項目深度介紹

## 報告存檔

```
reports/
├── 2026-04-20.md
├── 2026-04-27.md
└── ...
```

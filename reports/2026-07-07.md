## 個人精選

### [santifer/career-ops](https://github.com/santifer/career-ops) 🧠（本週排名：18）

- **它在做什麼：** 建構在 Claude Code 之上的 AI 求職系統，內建 14 種 skill 模式，涵蓋履歷／履歷 PDF 產出、職缺蒐集（web scraping）與批次處理流程，並附 Go 語言儀表板。
- **跟我的方向有什麼關係：** 直接對應「可整合履歷 / 個人資料的工具」這個標記——它把履歷生成、職缺篩選整套流程封裝成可重複執行的 agent skill，是個人 AI 助理處理「求職／履歷」這類個人資料場景的具體案例。
- **值得注意的技術選擇：** 用多個獨立 skill 模式（而非單一巨大 prompt）拆解求職流程各階段，並用 Go 撰寫儀表板而非常見的 Python/JS，顯示作者刻意讓執行層與 agent 邏輯層分離，可作為「skill 模組化」設計的參考案例。
